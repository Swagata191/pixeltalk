# 🔢 PixelTalk // Digit Recognition Dashboard

PixelTalk is a magnificent, dark-themed **Convolutional Neural Network (CNN) digit classification dashboard** built with Streamlit, TensorFlow, and Plotly. The system is engineered to provide realtime inference and deep visual diagnostics of handwritten digits from the MNIST dataset.

The interface is wrapped in a high-performance **cyberpunk matrix layout** (using deep blues, dark purples, and electric cyans) complete with an interactive canvas background that brings neural network outputs to life with responsive pixel-burst animations.

---

## 🌌 Core Features

- **Structural Grid Analytics Architecture**: Locks into a robust, jitter-free 3-column macro framework providing an immediately usable dashboard console.
- **Deep Cognitive Arbitration Diagnostics**:
  - `Bitstream Integrity Donut`: Displays the precise network duty cycle and focus node.
  - `Density Heatmap`: Illustrates the 2D entropy grid of activation sequences over a top-down mapped 2×5 spatial orientation.
  - `Probability Array Bars`: Visually weights probability margins with multi-tonal alpha mapping.
  - `Frequency Radar`: Plots high-dimensional vector alignments natively.
- **Micro-Interactions**: Hover states, smooth canvas translations, and glow enhancements on prediction vectors.

## 🛠️ Installation & Setup
Ensure you have a Python environment initialized (Python 3.9 - 3.11 recommended).
Clone or Download the Repository.
```
git clone <repository-url>
cd pixeltalk-core
```
Install Dependencies
```
pip install -r requirements.txt
```
Mount and Launch the System Core
```
streamlit run app.py
```

## 🧠 Neural Architecture Summary

The underlying system processes a standard normalized discrete matrix array ($28 \times 28 \times 1$ grayscale input) through a layered feature extraction pipeline:
- Convolutional Layer: Local spatial pattern and line weight detection.
- MaxPooling Layer: Invariance dimensionality reduction.
- Dense/Fully-Connected Head: Final Softmax routing to evaluate activation probability arrays across 10 output nodes (digits 0–9).
